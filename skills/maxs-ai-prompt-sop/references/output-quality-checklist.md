# 麦壳思 AI 生成图质量自查清单

学生和老师在 AI 生成完图、想判断"这张能不能用"时,逐项过这个清单。

---

## 通用清单(所有类型的 AI 图都要过)

### 基础合格线(任何一条不过就是废图)

- [ ] 主体清晰,没有"半成品""残片""幽灵物体"
- [ ] 没有畸形手指、错位结构、多余肢体(如有角色)
- [ ] 文字部分(如有招牌/书页)没有乱码
- [ ] 透视基本正确,没有"两个透视拼贴"的诡异感
- [ ] 光照逻辑一致,不出现"左边晨光右边夕阳"

### 风格合格线

- [ ] 风格符合提示词指定的目标游戏/艺术方向
- [ ] 没有风格漂移(如要中式出了日式、要赛博出了蒸汽朋克)
- [ ] 整体材质语言统一,不同物件不是各自不同的风格

### 用途适配线

- [ ] 比例符合用途(竖屏不能用作横屏封面)
- [ ] 分辨率足够(教学素材至少 1080p、印刷至少 4K)
- [ ] 关键信息没有被裁切

**任何一项不过 → 调提示词重抽,不要将就**

---

## 资产陈列图专用清单

### 白底版

- [ ] 背景确实是白色(不是浅灰),用取色器验证 #F8F8F8 以上
- [ ] 没有接触阴影、地面、环境反射
- [ ] 物件保留原色,没有被环境光染色
- [ ] 物件之间有清晰间距,不重叠粘连
- [ ] 大中小排布有节奏感,不是平铺
- [ ] 每个物件都是完整的,可以裁剪出来单独使用
- [ ] 数量大致符合提示词要求(±30% 内可接受)

### 深色氛围版

- [ ] 地面材质符合场景(湿漉漉的水泥 / 古老木地板 / 等)
- [ ] 投影方向一致(所有物件的影子指向同一方向)
- [ ] 氛围光不会盖过物件本身的细节
- [ ] 物件边缘有轻微环境光描边,显得"在场景里"而不是"贴上去的"

---

## 单件 3 视图专用清单(给 Tripo / Hunyuan3D 喂料)

- [ ] 三个视图严格保持同一物件(不是三个相似但不同的物件)
- [ ] 视角是正确的正面 / 侧面 / 3/4,不是相似的旋转角度
- [ ] 三视图比例一致,大小相同
- [ ] 物件完整可见,没有被画面边缘裁切
- [ ] 背景纯白,可以直接被 3D 重建工具识别为前景
- [ ] 没有阴影投在白底上(会被识别为额外几何体)
- [ ] 材质和颜色三视图之间一致,不出现"正面是木头侧面是石头"

---

## 概念美宣单图专用清单

- [ ] 有明确的视觉中心,不是均匀铺满细节
- [ ] 构图遵循一种构图法(黄金分割 / 三分法 / 中心对称 / ...)
- [ ] 有前景、中景、远景的层次
- [ ] 色彩有主调,不是花花绿绿全饱和
- [ ] 有叙事感(看一眼能想象出一个故事场景)
- [ ] 适合作为 portfolio 头图 / 项目封面 / 美宣 thumbnail

---

## 教学素材专用额外清单

(除了上面对应类型的清单,还要满足这些)

- [ ] 学生看一眼能看明白这是讲什么的
- [ ] 不依赖讲解就能识别图中所有元素
- [ ] 没有违反课程目标的元素(如教 modular 拼接,图里不能全是 unique 大物件)
- [ ] 可以被学生抄袭复现(提示词五变量能套出类似效果)
- [ ] 配套有 cheat sheet 或提示词卡片

---

## 客户/对外发布专用额外清单

(除了上面对应类型的清单,还要满足这些)

- [ ] 没有可识别的版权角色/IP(米奇、皮卡丘、马里奥、奇异博士等)
- [ ] 没有可识别的真实品牌 logo(可口可乐、耐克、Adobe 等)
- [ ] 没有可能引发争议的元素(政治符号、宗教争议、敏感地标)
- [ ] 风格不能让人误以为是某个具体游戏的官方泄露图(避免侵权风险)
- [ ] 如果用于商业场景,确认 AI 模型的商用授权(Banana Pro / Midjourney 商用规则不同)

---

## 自查后的决策树

**全过** → 入库 + 走复用包模板填写

**只在"教学/客户额外清单"不过** → 看用途,如果不对外发布可以用

**风格/用途适配不过** → 改提示词的风格锚点或用途变量

**基础合格线不过(畸形/残片/乱码)** → 直接 reroll,不要试图修

**反复 reroll 5 次还是不过** → 
1. 不是你写错,是模型本身搞不定这个组合
2. 换模型(Banana Pro → Flux / Midjourney)
3. 或拆分任务(把一张大图拆成多张小图)
4. 或图生图(找参考图垫底)

---

## 给学生的简化版(贴在作业单上)

\`\`\`
作业图自查 5 步:
1. 主体清晰、无残缺、无畸形? ✓ / ✗
2. 风格符合目标? ✓ / ✗
3. 用途适配(比例/分辨率)? ✓ / ✗
4. 提示词可复用、可教学? ✓ / ✗
5. 自己看一眼,合上电脑还想看下一张? ✓ / ✗

5 个✓ 才能提交。
\`\`\`

最后一条来自 MAXs 录播课 SOP 的"合上电脑还想不想看下一节"判断标准,在 AI 图的场景下同样适用。
