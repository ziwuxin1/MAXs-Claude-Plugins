## 模板 1:资产陈列图(白底版)【S 级,已三步迭代】

### 用途
教学素材 / 学生作业模板 / 3D 重建参考图(可裁剪单件后喂 Tripo)

### 模板

```
把图中的[场景类型]结构部件按照3D资产图一次摆放,
[数量]个结构部件不能重复,按照大中小依次排放,
8k分辨率,顶级摄影照片,俯视35度视角,
电影级布光,[光照方案],
风格化PBR材质,半写实搭配法线贴图层次,
辅以手绘质感的[材质细节],
保持物件比例关系准确,
[目标风格],asset showcase陈列图,白背景。无阴影
```

### 验证记录
- 模型: Banana Pro
- 验证次数: 3 次(雪山古建×2、街角咖啡馆×1)
- 成功率: 80%(5 次抽卡 4 次合格)
- 验证时间: 2026 年 3 月

### 变量库

**[场景类型]**
- 雪山古建场景 / 江南水乡集市 / 武侠客栈 / 日式神社 / 中式园林
- 赛博朋克街区 / 末日废土加油站 / 蒸汽朋克工坊 / 太空舱内饰 / 地下避难所
- 中世纪集市 / 维多利亚街景 / 北欧木屋 / 西部荒野小镇 / 美式郊区

**[数量]**
- 8-12 件:适合精修单品展示
- 15-20 件:甜蜜点,质量与数量平衡
- 25+ 件:容易出 bug,慎用

**[光照方案]**
- 黄昏暖色侧光搭配冷色环境光(国风/写实最佳)
- 正午顶光搭配阴影补光(写实白模感)
- 月夜冷光带霓虹边光(赛博朋克专用)
- 阴天柔和散射光(北欧极简/日系)
- 火光暖色主光配深蓝环境光(地下/末日)

**[材质细节]**
- 磨损、积雪、冰挂、风化(雪山题材)
- 锈蚀、霓虹反光、油污、电线(赛博)
- 苔藓、藤蔓、潮湿、青砖斑驳(江南/园林)
- 灰尘、裂纹、年代感、做旧(中世纪)
- 烧焦、弹孔、辐射痕迹、变形(末日废土)

**[目标风格]**
- 黑神话悟空 / 燕云十六声 / 卧龙苍天陨落(国风写实)
- 赛博朋克2077 / 边缘行者 / 攻壳机动队(赛博)
- 暗黑破坏神4 / 流放之路2(暗黑写实)
- 原神 / 鸣潮 / 崩坏星穹铁道(卡通渲染)
- 荒野大镖客2 / 最后生还者(西部/末日写实)
- 守望先锋 / 风暴英雄(暴雪风格化)

### 骨架句(不能动)
- `按照大中小依次排放`
- `保持物件比例关系准确`
- `白背景。无阴影`(必须放最后)

### 踩坑提醒
- 不要加 `RGB(255,255,255)`、`无影灯`、`产品摄影` —— 这些会让模型过度规训,出图僵硬
- 不要在中间出现"白底"指令 —— 必须放最后才稳
- 黄昏光不会污染白底,放心用 —— 反直觉但已验证

## 模板 2:单件 3 视图(给 Tripo/Hunyuan3D 喂料)【A 级】

### 用途
单件 3D 重建参考图,白底无阴影,正面 + 3/4 + 侧面

### 模板

```
单个[物件名称],三视图展示,
正面视图、3/4 前视图、侧面视图,
纯白背景,无阴影,无地面,
柔和均匀的产品级布光,中性白光,
保留物件原始材质和颜色,
[风格关键词]风格,
8k 高清,游戏资产参考图,turnaround sheet
```

### 验证记录
- 模型: Banana Pro / Midjourney v6
- 推荐配合 Tripo 3.0 或 Hunyuan3D-2.0 重建
- 注意: 三视图模式 Midjourney 比 Banana Pro 更稳

### 关键技巧
- 物件名称要具体到部件级,不要写"建筑",要写"中式飞檐屋顶 single roof unit"
- 加 `concave side facing camera` 或 `convex side facing camera` 控制弯曲方向(如香蕉、屋顶)
- 加 `subsurface scattering reference` 用于后续 SSS 材质制作(冰、玉、皮肤、果实)

## 模板 3:深色氛围陈列图(美宣 / portfolio)【S 级】

### 用途
B站 / 小红书封面、80 Level 投稿、portfolio 项目页、客户提案

### 模板

```
把图中的[场景类型]结构部件按照3D资产图一次摆放,
[数量]个结构部件不能重复,按照大中小依次排放,
8k分辨率,顶级摄影照片,俯视35度视角,
电影级布光,[光照方案],
风格化PBR材质,半写实搭配法线贴图层次,
辅以手绘质感的[材质细节],
保持物件比例关系准确,
[目标风格],asset showcase陈列图,深色地面带轻微反射
```

### 关键差异
- 删掉 `白背景。无阴影`
- 加 `深色地面带轻微反射` 或 `深色木质地面 / 哑光黑色地面 / 深灰水泥地面`
- 保留氛围光指令(黄昏/月夜等),光会照在地面上加强氛围

### 验证记录
- 已用在 MAXs 雪山古建系列产出
- B站测试封面 CTR 比白底版高 40%
- 适合"晒图"场景,不适合"教学"场景
