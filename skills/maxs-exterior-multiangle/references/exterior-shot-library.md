## 场地身份保持句(殿后用，每条原样复制)

以废料场案例为例:

> 场景还是参考图里同一片废料场:左侧多层锈蚀塔楼(墙面有涂鸦痕迹)、右上方悬在场地上空的巨型黄色环形机械残骸、中央锈蚀电线杆和斜拉线缆、深处的拱形机库门洞、右侧集装箱和机车残骸堆，地面是沙土混金属散落物。锈橙+沙黄+军绿的色调、过曝发白的正午晴天、四周延续的废土荒地全部保持一致，但不要复刻参考图的构图。

要点:地标带方位(左侧/右上/中央/深处)、涂鸦只写"痕迹"不写文字、天空状态和环境边界都要锁。

## 太阳方位句(每条按机位重写)

先从基准图影子反推太阳方位，写成一句基准事实，例:"太阳在场地的右上方向(基准图影子落向左下)"。然后每个机位翻译:

| 机位相对基准图 | 太阳方位句写法 |
|--------------|--------------|
| 同向(侧打/推近) | 太阳仍在画面右上，影子落向左下 |
| 反打(转180°) | 太阳转到画面左前方，结构影子朝镜头方向拉长，逆光局部眩光 |
| 高角度俯瞰 | 太阳在右上，所有结构的影子统一落向左下，影子成为俯视构图的方向线 |
| 低角度仰拍 | 太阳在结构后方/侧后方，边缘形成亮边，镜头方向天空过曝 |

## 完整范例(废料场·反打，机位前置格式)

> 把镜头放到场地深处的拱形机库门洞前，回头看向参考图镜头所在的方向——这张图的构图必须和参考图明显不同:参考图看向场地深处，这张反过来看场地入口方向，能看到参考图中背对我们的那一侧:锈蚀塔楼的另一面、黄色环形残骸的背面支撑结构，延续相同的废土结构风格、不新增大型建筑，远处是空旷的废土地平线。此机位下太阳转到画面左前方，各结构的影子朝镜头方向拉长，画面带逆光眩光。场景还是参考图里同一片废料场:左侧多层锈蚀塔楼(涂鸦痕迹)、巨型黄色环形机械残骸、中央电线杆和拉索、集装箱堆、沙土地面，锈橙+沙黄+军绿色调和过曝白天空全部一致，但不要复刻参考图的构图。

## 操作流程

1. 同一对话上传基准图;先从影子反推太阳方位记下来
2. 顺序:侧打 → 反打 → 高角度俯瞰 → 远景 → 仰拍 → 特写(先积累场地布局，再做脑补量大的机位)
3. 每机位抽 2-4 张;构图雷同弃;香蕉 2 跑量、Pro 精修
4. 漂移时回完整保持句+重传基准图

## 翻车应对速查(按症状)

- **整组复刻基准图** → 机位句没前置;补构图差异声明
- **侧打走不动(开阔场景复制偏置强)** → 用**近/远法**(继承 maxs-multiview-reference 镜像坑结论):点名"参考图中位于X侧的XX现在离镜头最近、占据前景，只能看到它的侧面;参考图中X侧的XX现在在画面最远处"，并加"参考图的正面布局不应该再出现"(2026-06-10 废料场实测:光说"横着看"必复刻)
- **影子方向乱** → 该条没写影子推演;按"同一个太阳"逐机位翻译
- **远景冒出城市/雪山** → 保持句锁边界"四周是延续的废土荒地，地平线上没有城市和山脉"
- **地标换位/增殖** → 地标带方位写进保持句;先侧打积累布局
- **正午变黄昏** → 锁天空状态;黄昏只作为独立氛围机位单独要
- **涂鸦出乱码** → 只写"涂鸦痕迹"，不写具体文字
- **巨型结构失去压迫感** → 写参照物("残骸直径约等于塔楼三层高")

## 学生作业三查清单

- [ ] 地标查:大型结构的形状、数量、相对位置与基准图一致
- [ ] 影子查:全图影子能用同一个太阳解释通
- [ ] 边界查:环境延续世界观、天空天气一致，没冒出新天际线
