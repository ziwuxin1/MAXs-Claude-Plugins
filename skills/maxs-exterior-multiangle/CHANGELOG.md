# Changelog

## 1.0.0 (2026-06-10)

首个版本。基于废土废料场概念图（锈蚀塔楼+巨型黄色环形残骸）实测开发。

### 核心方法
- 与室内多机位的三个本质差异：太阳+影子是最硬锚点（逐机位重推影子落向）、开放边界必须显式锁定（防地平线脑补城市/山脉）、多出无人机高角度与远景大全景机位
- 黄金公式继承室内 skill 机位前置版：机位句前置 → 构图差异声明 → 太阳方位与影子推演 → 场地身份保持句殿后 → "不要复刻构图"收尾
- 九机位指令库（含黄昏氛围变体——唯一允许改太阳的机位）+ 太阳方位句翻译表 + 室外三查（地标/影子/边界）

### 实测沉淀（2026-06-10 废料场案例）
- **侧打走不动坑**：开阔场景复制偏置比室内强，光说"横着看"必复刻——用近/远法点名"参考图中X侧的XX现在离镜头最近、占据前景"+"参考图的正面布局不应该再出现"
- 涂鸦只写"痕迹"不写文字（继承系列原则）
- 尺度锚点写参照物（"残骸直径约等于塔楼三层高"）
