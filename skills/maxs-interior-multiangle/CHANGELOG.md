# Changelog

## 1.0.0 (2026-06-10)

首个版本。基于佛殿概念图（鎏金观音大殿）实测开发，整组 6 机位（定场/反打/侧打/仰拍/回廊俯瞰/轴线推近）验证通过。

### 核心方法
- 与单物体多视图的本质区分：转镜头不转物体，保留轴=空间身份（结构+光源逻辑+英雄物方位）
- 黄金公式（机位前置版）：机位句前置 → 构图差异声明 → 逐机位光源推演 → 空间身份保持句殿后 → "不要复刻参考图的构图"收尾
- 八机位指令库 + 佛殿完整范例 + 室内三查（空间/光向/主体）

### 实测沉淀（2026-06-10 佛殿案例）
- **"保持句压死机位句"坑**：保持句在前导致 5 机位中 3 个原样复刻基准图；机位前置改版后 5/5 一次通过——顺序是生死线
- **交付硬规范**：每条提示词必须完整自含，禁止"……/同样的要求"省略式交付（实测用户只复制了保持句导致原图复刻）
- 逐机位光源推演是室内场景的命门：反打=逆光剪影、仰拍=天光入镜，每条单独写
- 特写与远景细节冲突时以特写为准反喂统一
