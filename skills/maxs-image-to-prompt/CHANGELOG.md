# Changelog — maxs-image-to-prompt

## v1.1.0 (2026-05-22)

默认二创模式校准:从"保骨架换题材"改为**"保题材·保骨架·保构图,靠镜像/换机位/换视角/换尺度产出同场景的一组变体"**。

- 默认模式① = 同场景多机位/镜像变体(锁题材+构图+冷暖对冲,变镜头)
- "换题材"降级为模式③(激进可选,深海版即此例)
- 保留轴扩到"场景身份"(题材+内容+冷暖对冲+布局);可变轴开大到镜像/机位/视角/尺度
- 同步更新 derivation-modes.md、case-ripperdoc.md

## v1.0.0 (2026-05-22)

首版。麦壳思「图转提示词·二次创作」SOP。

- 核心定位:从一张参考图出发,提取构图 DNA → 锁保留轴 + 换可变轴 → 生成二次创作提示词
- 默认模式:保骨架换题材(经赛博黑诊所→深海研究站案例实测确认幅度)
- 两套 JSON schema:场景图 / 资产陈列图(陈列图直接对接 maxs-ai-prompt-sop 五变量黄金模板)
- 四种二创模式:保骨架换题材 / 换氛围 / 换镜头 / 系列变体
- references:json-schema.md、derivation-modes.md、case-ripperdoc.md
- 与 maxs-ai-prompt-sop 共用三步迭代、分级体系、复用包、抽卡规范
