# Changelog

## 2.0.0 (2026-06-11)

重定位重构。基于佛殿→帝王陵墓地宫的多轮真实迭代沉淀。

### 重定位
- 铁律分流：目标图存在参考图就走图生图多机位三件套；本 skill 只管"目标图不存在"的两件事——换题材迁移(模式A)、起点反推(模式B)
- v1 的同场景多机位/换氛围/系列变体模式移交 multiview/interior/exterior 三件套
- 实测依据：文字复现同一场景，氛围/材质/光照全面劣于原图

### 新增铁律
- **语境锁死**：古代进古代出、赛博进赛博出；跨语境迁移仅当用户显式点名，技能不得自行提议(用户规则)
- **挂图迁移**：香蕉系迁移把原图作参考图上传，骨架由图保证，文字只换皮肉+锁语境
- **定稿模板**(用户多轮实测确认)：骨架声明前置+替换清单+语境锁+自然句收尾，无 tag 尾巴
- **抽卡纪律**：单点失败先同提示词重抽 2-3 次再考虑改词；补丁一次只加一个；终版比中间版短——实测"修正版"堆约束后全面劣于原版，过度纠正是最大的坑
- **最小变更链**(精修阶段)：满意图定为新基准，"只做一件事"式指令逐维微调

### 结构
- 完整 JSON schema 简化为两层"骨架卡"(骨架层/题材层，含时代语境字段)
- references：新增 skeleton-card.md(模板+同语境/跨语境双案例)，废弃 json-schema.md、derivation-modes.md、case-ripperdoc.md(均为 v1 文生图时代产物)
