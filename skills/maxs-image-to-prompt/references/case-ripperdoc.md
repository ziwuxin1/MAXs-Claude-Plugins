## 案例名称
赛博朋克地下黑诊所手术室 → 换题材二创

## 原图 DNA(场景图 JSON)
见 `json-schema.md` 的"填写示例",此处略。核心 DNA:
- 广角定场,双焦点构图
- 中央英雄道具(血迹手术椅)被顶部环形手术灯打亮
- 左侧墙体破开暖红光缺口,内有吊挂物 + 剪影人物
- 冷绿青雾光 vs 左侧暖红光,强冷暖对冲
- 满地叙事杂物(发电机、盘绕软管、垃圾)

## 拆轴
- **保留轴(锁死)**:广角定场 / 中央英雄道具被环形光打亮 / 左侧暖光缺口透剪影 / 全局冷暖对冲 / 满地叙事杂物
- **可变轴(换掉)**:题材(赛博黑诊所)、英雄道具(手术椅)、氛围(body horror)、风格锚点

## 用的模式
③ 换题材(激进模式) —— 注意:本案例演示的是"换题材"这个**激进可选模式**。本 skill 的**默认模式是①同场景多机位/镜像变体**(锁题材+构图,只换镜头/镜像/视角)。深海版是用来说明"构图迁移到新世界"长什么样,不是默认幅度。

## 最终二创提示词(变体②·深海研究站版,已被确认 OK)
```
一座废弃深海研究站内部改造的手术室,电影级广角定场镜头。
画面中央一把锈迹斑斑的旧检查椅,数条老旧机械臂从顶部环形灯下垂。
四周是泛黄的旧仪表盘与闪烁的监测屏。左侧船体破开一个透出暖橙色警示灯光的裂口,
海水与雾气涌入,裂口处立着一个剪影人物,与舱内幽冷的蓝绿海光形成冷暖对冲。
地面是湿滑的金属格栅,散落缠绕水管、氧气瓶、杂物。体积水雾,高对比暗部,
海水锈蚀与盐渍质感,写实PBR,8k,电影级布光,幽闭压抑的深海惊悚氛围。比例16:9。
```

**注意骨架句的"换名词不换句式"技巧:**
- 原图:`左侧墙体破开一个透出暖红光的缺口,里面立着一个剪影人物`
- 二创:`左侧船体破开一个透出暖橙色警示灯光的裂口,裂口处立着一个剪影人物`
- 句式骨架完全一样,只换了"墙体→船体""暖红→暖橙警示""缺口→裂口"——这就是为什么"题材全换了还认得出同源"。

## 模型 / 参数
- 模型:Nano Banana / Banana Pro / Gemini
- 比例:16:9

## 二创幅度反馈(含一次重要校准)
- 第一次纯反推(没拆轴)→ 出图"太像原图",几乎是克隆,只能用来验证 DNA 抽对没
- 试了"换题材"(深海版)→ 题材完全不同、构图一眼同源 → 这是**模式③(激进)**的样子,好用但不是默认
- **校准结论**:默认二创应是**模式①——保题材·保构图,靠镜像/换机位/换视角/换尺度产出同场景的一组镜头**。题材和场景内容守死,变的是"怎么拍"。换题材(③)是想把构图迁移到新世界时才用。
- 一句话:默认是"同一个场景的不同镜头",不是"同一个构图的不同题材"。

## 踩坑提醒
- 原图屏幕/墙上的文字(CYBERPSYCHOSIS WARNING 等)不要试图复现,模型拼不对,只描述"警示标识感"
- 信息过载:原图杂物极多,二创时次要杂物概括成"散落杂物",别一件件塞
- 冷暖对冲是这张图的命,二创时必须把"暖光缺口 vs 冷调主体"明确写进提示词,丢了就垮
