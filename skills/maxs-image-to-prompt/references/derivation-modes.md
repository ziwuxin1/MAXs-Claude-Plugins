# 二次创作四模式 + 拆轴清单

二创的核心是把 JSON 字段分成**保留轴(锁死)**和**可变轴(放开)**,再选模式决定锁多少、换多少。

**默认理念:锁住"场景身份"(题材+构图DNA),变的是"怎么拍它"。** 同一个场景换镜头/镜像/视角,产出一组画面,而不是一张。把构图整个搬到新题材,是更激进的可选模式③,不是默认。

## 拆轴:保留轴 vs 可变轴(默认模式)

| JSON 字段 | 默认归类 | 说明 |
|----------|---------|------|
| 场景类型 / 题材 | **保留轴** | 默认守死,这是"还是这个场景"的根 |
| 关键道具 / 英雄道具 | **保留轴** | 场景内容不变 |
| 光照方案(冷暖对冲关系) | **保留轴** | 一张图最值钱的 DNA,守死 |
| 空间布局 / 叙事元素 | **保留轴** | "侧向暖光缺口透剪影"这类结构守死 |
| 氛围关键词 | **保留轴**(默认) | 默认不动;要动就走模式② |
| 构图.镜头(机位/视角) | **可变轴** | 默认靠它出变体:俯仰、推拉、低角度 |
| 镜像翻转(左右) | **可变轴** | 把左侧元素镜像到右侧,出对称变体 |
| 焦距 / 尺度 | **可变轴** | 广角定场 ↔ 英雄道具特写 |
| 光线时间(可选) | **可变轴** | 想要"同场景不同时刻"再动,否则锁 |

> 关键技巧:默认模式下,**场景描述整段沿用,只改机位/镜像/视角的措辞**。镜像就是"左↔右"对调,其余文字不动。

## 四种模式

### ① 同场景多机位/镜像变体(默认 · 已校准甜蜜点)
- **锁**:题材 + 场景内容 + 冷暖对冲 + 空间布局(场景身份全锁)
- **换**:镜像翻转、机位/视角(俯仰、左右移动、推远拉近)、焦距与尺度(广角定场↔特写)
- **用途**:给同一个场景/关卡出一组 beauty shot、分镜、封面图+细节图
- **幅度校准**(2026-05-22 确认):做出来要"一眼还是这个场景,但是另一个镜头/角度"。变体之间能凑成一套,不是各画各的。

### ② 换氛围
- **锁**:题材 + 构图都留
- **换**:只换氛围基调(惊悚 body horror → 高冷科技 / 废墟 → 运作中 / 白天 ↔ 夜晚)
- **用途**:同一场景的不同状态、不同时间(如诊所"废弃"vs"运作中")

### ③ 换题材(激进 · 可选)
- **锁**:构图骨架(镜头结构、英雄道具的位置和打光、冷暖对冲)
- **换**:题材整个换(赛博 → 深海/国风/废土)、英雄道具、风格锚点、材质
- **用途**:举一反三、把一个验证过的好构图迁移到新世界(深海研究站版就是这个模式)
- **技巧**:骨架句句式不动,只换里面的名词(墙体→船体、暖红→暖橙警示)

### ④ 系列变体
- **锁**:题材 + 构图 + 氛围都留
- **换**:只换细节道具、微调布局
- **用途**:高度统一的成套图(如同一诊所的不同角落)

## 模式选择速查

- 想"同一个场景出一组镜头/分镜/封面" → ①(默认)
- 想"同场景不同状态/时间" → ②
- 想"把这个构图迁移到全新题材" → ③
- 想"风格高度统一的成套小图" → ④

## 二创幅度自检(两个反向坑)

- **变体没变(默认模式最常见)** → 机位/镜像没真动,出了张几乎一样的。明确写新机位:低角度、镜像左右、推近拉远、俯视。
- **场景身份崩了(走③时)** → 题材换太猛,认不出同构图。回到保留轴清单,把构图骨架句原样写回,只换名词。
